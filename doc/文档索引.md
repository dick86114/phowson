# Phowson 项目文档体系

本文档目录包含 Phowson 项目的全套开发与运维文档。

## 1. 项目管理 (Project Management)
- [产品需求文档 (PRD)](./项目管理/产品需求文档.md): 包含业务目标、功能需求、非功能需求及变更管理流程。
- [任务分解与工时表](./项目管理/任务分解与工时表.md): 详细的任务分解、工时预估与资源分配。
- [里程碑计划](./项目管理/里程碑计划.md): 关键节点交付物与验收标准。
- [待办事项 (Backlog)](./项目管理/待办事项.md): 详细的任务清单与迭代计划。
- [路线图 (Roadmap)](./项目管理/路线图.md): 项目长期规划与版本迭代路径。

## 2. 技术开发 (Technical Development)
- [技术架构与规范](./技术开发/技术架构与规范.md): 系统架构、数据库设计、代码规范与核心算法。
- [API 接口文档](./技术开发/API接口文档.md): 详细的接口定义与调用示例。
- [移动端 SDK 开发约定](./技术开发/移动端SDK开发约定.md): 移动端开发约定。

## 3. 运维部署 (Operations & Deployment)
