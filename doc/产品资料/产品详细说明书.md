# 产品详细说明书 (Product Specification)

## 1. 产品信息
- **产品名称**: Phowson (浮生)
- **版本**: v1.0
- **适用平台**: Web (Chrome/Edge/Safari/Firefox), Mobile Web (iOS/Android), PWA
- **开发商**: Phowson Team

## 2. 功能特性清单

### 2.1 基础功能
- **多格式支持**: JPG, PNG, WEBP, HEIC (需转码)。
- **EXIF 解析**: 自动提取光圈 (F)、快门 (S)、感光度 (ISO)、焦距、相机型号、镜头型号、拍摄时间、GPS。
- **原图存储**: 支持保留原始分辨率图片，同时生成缩略图用于预览。

### 2.2 AI 功能
- **模型支持**: Google Gemini Pro Vision / OpenAI GPT-4o。
- **能力**: 图像内容理解、OCR 文字识别、风格分析、评分建议。

### 2.3 管理功能
- **RBAC 权限**: 支持 Admin (完全控制)、Family (查看/评论)、Guest (仅查看)。
- **批量操作**: 批量上传、批量删除、批量修改分类。
- **数据导出**: 支持导出所有照片元数据为 JSON/CSV。

## 3. 技术参数
- **前端架构**: React 18 + TypeScript + Vite + Tailwind CSS
- **后端架构**: Node.js (Fastify) + PostgreSQL 15 + pgvector
- **存储**: Local Filesystem / AWS S3 / Cloudflare R2 / MinIO
- **部署方式**: Docker Compose 一键部署

## 4. 系统要求

### 4.1 服务器端 (Server)
- **CPU**: 2 Core+ (推荐 4 Core)
- **RAM**: 2GB+ (推荐 4GB, 若运行本地 AI 模型需更高)
- **Disk**: SSD 50GB+ (取决于照片数量)
- **OS**: Linux (Ubuntu/Debian/CentOS) / macOS

### 4.2 客户端 (Client)
- **浏览器**: Chrome 90+, Safari 14+, Edge 90+, Firefox 90+
- **网络**: 建议 4G/5G/Wi-Fi 环境

## 5. 售后服务
- **开源版**: 社区支持 (GitHub Issues / Discussions)。
- **商业版**: 提供 7x24 小时技术支持、专属部署服务与定制开发。
