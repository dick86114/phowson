# 用户操作手册 (User Manual)

## 1. 快速入门

### 1.1 系统登录
1. 访问系统首页 `http://your-domain.com`。
2. 点击右上角 "登录" 按钮。
3. 输入注册邮箱和密码，或使用第三方账号 (Google/GitHub) 登录。

### 1.2 界面概览
- **顶部导航栏**: 包含 "首页"、"地图"、"相册"、"上传" 等入口。
- **主区域**: 展示照片瀑布流或地图聚合点。
- **侧边栏 (移动端)**: 包含个人设置、关于页面等。

## 2. 核心功能操作

### 2.1 上传照片 (Upload)
1. 点击导航栏 "Upload" 图标。
2. 拖拽照片到上传区域，或点击选择文件 (支持 JPG/PNG/WEBP)。
3. **AI 智能填单**: 系统会自动识别照片内容，生成标题、描述和标签。
4. **人工校对**: 检查 AI 生成的内容，如有误可手动修改。
5. 点击 "发布" 按钮完成上传。

### 2.2 浏览与搜索
- **瀑布流浏览**: 在首页向下滑动，系统会自动加载更多照片。
- **详情查看**: 点击任意照片进入详情页，可查看 EXIF 信息 (光圈、快门、ISO)、拍摄地点及 AI 点评。
- **地图模式**: 点击 "Map" 进入地图模式，缩放地图查看不同地点的拍摄足迹。
- **搜索**: 在顶部搜索框输入关键词 (如 "猫", "日落")，支持自然语言搜索。

### 2.3 互动与社交
- **点赞**: 点击照片下方的 "心形" 图标。
- **评论**: 在详情页底部输入评论内容并提交。
- **分享**: 点击 "分享" 按钮，生成精美海报或复制链接分享给好友。

## 3. 常见问题 (FAQ)

- **Q: 为什么上传的照片没有 GPS 信息?**
  - A: 请确保拍摄时已开启相机的定位功能，且上传原图未经过微信等压缩处理。
- **Q: 如何修改已发布的照片信息?**
  - A: 在详情页点击 "编辑" 按钮 (仅限本人或管理员)，修改后保存即可。
- **Q: 忘记密码怎么办?**
  - A: 在登录页点击 "忘记密码"，通过注册邮箱重置。

## 4. 使用技巧

- **快捷键**: 在详情页使用键盘 `←` `→` 键可快速切换上一张/下一张。
- **PWA 安装**: 在移动端浏览器点击 "添加到主屏幕"，可像原生 App 一样离线浏览。
