## 3. 进度跟踪机制

- **每日站会 (Daily Stand-up)**: 每天 10:00，全员参加，时长 15 分钟。同步昨日进展、今日计划及遇到的阻碍。
- **周报 (Weekly Report)**: 每周五 17:00 前提交，汇总本周完成任务、工时消耗及下周计划。
- **进度看板 (Kanban)**: 使用 GitHub Projects / Jira 管理任务状态 (To Do, In Progress, Review, Done)。
- **风险预警**: 任何任务延迟超过 20% 需立即上报项目经理，启动应对措施。
